Azure Upload and Download

This is the home page for a sample application used to illustrate the upload and download of files using the Azure Blob Storage REST API.


Upload or Download a file:

Upload a video to the Azure Blob Storage imsstorage private container. The file is uploaded using the Azure Blob Storage REST API.
The API can be called using applications and languanges that can make HTTP requests, like Java or Javascript.

Download a video from the Azure Blob Storage imsstorage private container. The file is downloaded using the same REST API.
The file will be downloaded to a local directory.


Streaming and Non-Streaming Video:

  • MPEG-4 (MP4) Video. A MP4 video in the public container.
  • MPEG-DASH (MPD) Video. A manually transcoded video in the public container.
  • MPEG-4 (MP4) CDN Video. A MP4 video in Azure CDN.
